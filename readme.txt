Updates all urls and content links in your website.

== Description ==

Facing an issue during the transfer of your website from one server to another server with the same URL..? You're at the right place where you need to install a OneClick Search and Replace plugin and update your Old URLs to New URLs of your domain without changing into the database.

There are a few benefits to using this plugin such as,

1. Save your important time.
2. Single click server migration.
3. No need to do manual URLs update in the database.
4. Easy to use and compatible for developers and users.
5. Speed-up your project deployment process. 

= Features: =
*   User can choose to update links embedded in content, excerpts, custom fields, and custom options function.
*   User can choose whether to update links for attachments.
*   User can see updated URL details on the dashboard after submitting the request.


== Installation ==

Installation and uninstallation are very simple processes. You can use WordPress automatic installation or follow the manual instructions below.



= Installing: =

1. Download the OCSR package.
2. Extract it to the "plugins" folder of your WordPress directory.
3. In the Admin Panel, go to "Plugins" and activate it.
4. Go to Tools -> OCSR URLs to use it.
5. You can in the sidebar of the WordPress dashboard with the actual name of OCSR URLs.

= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Go to the "plugins" folder of your WordPress directory and deactivate the plugin.
3. If you don't want to use this plugin, you can permanently delete it.

= Usage: =

Using this plugin is very simple. Once it has been activated, navigate to tools -> OCSR URLs and follow the instructions. 

Please Note: Changes are irreversible. Before using this plugin, please backup your website before proceeding by any medium (Either by your hosting panel or any backup restore plugin).

== Frequently Asked Questions ==

= Why are my urls not updated? =

URLs are only replaced when an exact match is found. Be sure that you have entered the correct URL and press submit button. (Note: Matching is case-sensitive, please use as per your requirements.)

= Why do I see the message "You do not have sufficient permissions to access this page" ? =

Make sure that the plugin is activated and that you are an administrator level user access.
